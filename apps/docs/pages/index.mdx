# What is UniLLM?

UniLLM is a TypeScript library that enables you to interact with any LLM (Large Language Model) via a unified API - the OpenAI API, in a type-safe way.

## Getting Started

<Steps>

### Install UniLLM
```bash copy
npm i unillm
```

### Make an API call
<DynamicCodeExample>
```ts copy
import { UniLLM } from 'unillm';

/*
  #SETUP#
*/

// Setup UniLLM
const uniLLM = new UniLLM();

// Use any LLM provider and model
const response = await uniLLM.createChatCompletion("#MODEL#", {
  temperature: 0,
  messages: [
    {
      role: "user",
      content: "How are you?"
    }
  ]
})
```
